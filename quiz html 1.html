inline vs block*

Every html element has a default display value, depending on what type of element it is. The two most common
display values are block and inline.

Block-level elements

  • element always starts on a new line, and the brosers autimatically add some space(a margin)>
    before and after the element
  • a block-level element always takes up the full width avalible(stretches out to the left and right as far as it can)
